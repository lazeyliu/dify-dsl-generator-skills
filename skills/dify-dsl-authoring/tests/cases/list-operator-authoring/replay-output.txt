**模式判断**
这是 `workflow` 模式，目标是对数组做排序和截断后再输出。

**模板选择**
选择 `基础 Workflow 模板`，在中间插入 `list-operator` 作为列表处理节点。

**节点清单**
1. `start_node`
2. `code_node`
3. `list_node`
4. `end_node`

**连边清单**
1. `start_node -> code_node`
2. `code_node -> list_node`
3. `list_node -> end_node`

**字段检查表**
- `variable` 应引用数组来源
- `order_by` 结构应稳定存在
- `limit` 结构应稳定存在
- `end` 输出应引用 `list_node.result`

**DSL 草稿**
```yaml
data:
  type: list-operator
  variable: [code_node, result]
  order_by:
    enabled: true
    value: asc
  limit:
    enabled: true
    size: 2
```
