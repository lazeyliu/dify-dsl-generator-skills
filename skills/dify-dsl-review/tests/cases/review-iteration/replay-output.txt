**模式判断**
- 这是 `kind: app` + `app.mode: workflow` 的迭代型 workflow，主链路是 `code -> iteration -> end`。

**节点清单**
- `start_node`
- `code_node`
- `iteration_node`
- `iteration_start`
- `template_node`
- `end_node`

**连边清单**
- `start_node -> code_node`
- `code_node -> iteration_node`
- `iteration_start -> template_node`
- `iteration_node -> end_node`

**字段检查表**
- `iterator_selector` 指向 `code_node.result`
- `output_selector` 指向 `template_node.output`
- `start_node_id` 指向 `iteration_start`
- 容器外 `end` 输出引用 `iteration_node.output`

**风险分级**
- 阻塞项: 无
- 高风险项: 无
- 中风险项: 未做独立复核；数组元素类型变化可能影响模板输出
- 优化项: 可显式补更多 error_handle_mode 说明

**最终结论**
- `需要人工确认`
