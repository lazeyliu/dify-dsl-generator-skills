- 这是 `advanced-chat` 模式的知识检索问答链路，主链路是 `start -> knowledge-retrieval -> llm -> answer`。
- 结构判断：主路径清楚，但仍属于轻量 RAG 问答骨架。

**主要问题**
- `gate_blocked` 阻塞项：未做实体/模型级校验和独立复核，发布门禁不能判通过。
- `evidence_contract` 高风险：检索结果虽然进入了 `llm.context`，但没有额外观测字段记录检索命中与证据引用。
- `fallback_gap` 高风险：未显式处理检索为空的路径。

**发布门禁与覆盖率**
- 发布门禁状态：结构检查 `待定`，输入输出约定检查 `待定`，风险检查 `失败`，回归检查 `待定`，观测字段检查 `失败`，复核一致性检查 `待定`。
