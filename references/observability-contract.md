# 观测契约

用于复杂分支、工具链、fallback、多轮状态和高风险决策。

## 最小观测字段

关键决策节点优先提供以下字段中的适用项：

- `reason_code`
- `selected_path`
- `evidence_ref`
- `fallback_reason`
- `contract_version`
- `normalization_notes`
- `confidence_level`

## 规则

- 字段值优先稳定、短小、可枚举，不要写成长段自由文本。
- 观测字段默认用于内部调试，不直接展示给终端用户。
- 不要在观测字段中透传敏感原文、凭证、完整工具返回或不必要上下文。
- 最终结论若无法追溯到 `reason_code` 或 `evidence_ref`，应视为观测契约不足。

## 覆盖范围

至少覆盖这些节点或链路中的适用项：

- 关键分支决策
- fallback / 降级节点
- 外部工具或请求节点
- 关键状态写入节点
- 最终输出前的归一化或渲染节点
